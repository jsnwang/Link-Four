Link Four
Connect 4 in Java Swing

Click the buttons on top to drop pieces down the board.
First to 4 in a row wins.
Player name input is optional and you can only change it by starting a new game.
Color choice is optional and can be changed in preferences menu.

Known Bugs:
Cannot use keyboard after using buttons
New Game opens a new window the first time
Pieces do not drop correctly sometimes
Keyboard controls drop multiple pieces when you new game
